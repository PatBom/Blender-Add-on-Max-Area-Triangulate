# Blender Add-on: Max Area Triangulate

<img width="1116" height="779" alt="kuva" src="https://github.com/user-attachments/assets/d4f73806-20c8-4f4f-852b-46fa2fdc8900" />

Max Area Triangulation allows you to take a loop and fill it with an endcap using the Max Area method. It maximises the area of triangles within the endcap, avoiding the creation of long and narrow triangles that may impact performance.

Add-on inspired partially by Humus' performance analysis on the subject: https://www.humus.name/index.php?page=Comments&ID=228

To use Add-on:
  1. Download the MaxAreaTriangulation_AddOn file.
  2. Install and activate the Add-on in Blender.
  4. Select a loop in edit mode.
  5. Navigate to 3d viewport's side panel's Edit tab.
  6. Find "Max Area Triangulation" panel and press the "Triangulate" button.

To use Script:
  1. Download the MaxAreaTriangulation_Script file.
  2. Load the script in Blender's Scripting window.
  3. Select a loop in edit mode.
  4. Run the script.
